Návod na spuštění projektu (C# / .NET)
1. Příprava prostředí
Ujistěte se, že máte nainstalované Visual Studio 2022 (nebo novější) s nainstalovanou sadou funkcí Vývoj pro desktopové aplikace pomocí .NET.

Rozbalte projekt ze souboru .zip.

2. Otevření a sestavení
Otevřete soubor MudGame.sln ve Visual Studiu.

V horní liště klikněte pravým tlačítkem na Solution a zvolte Rebuild Solution, aby se stáhly případné balíčky a kód se zkompiloval.

3. Spuštění aplikace
V C# MUDu musíte spustit oba projekty najednou:

Klikněte pravým tlačítkem na Solution -> Configure Startup Projects...

Zvolte Multiple startup projects.

U projektů Server i Client nastavte akci na Start.

Potvrďte a stiskněte F5 (Start).
